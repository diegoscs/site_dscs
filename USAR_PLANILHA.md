# 📊 COMO USAR A PLANILHA PROSPECCAO_COMPLETA.csv

## ✅ PASSO 1: IMPORTAR PARA GOOGLE SHEETS

1. Abra: https://sheets.google.com
2. Clique em **"Novo"** → **"Planilha"**
3. Clique em **"Arquivo"** → **"Importar planilha"**
4. Escolha o arquivo: **PROSPECCAO_COMPLETA.csv**
5. Selecione **"Inserir nova planilha"**
6. Clique **"Importar"**

Pronto! Você tem a planilha de forma online.

---

## 📋 O QUE VOCÊ TEM:

```
ID              = Número sequencial
Nome Empresa    = Nome da empresa
Categoria       = Tipo de negócio
Bairro          = Localização em SP
Telefone        = Contato telefônico
Email           = Email (quando disponível)
Tem Site?       = ✅ ou ❌
Status Contato  = Não contactado / Enviado / Respondeu / Fechou
Data Contato    = Quando você enviou
Resposta        = O que responderam
Observações     = Detalhes importantes
Ticket          = Preço estimado do projeto
Prioridade      = Alta / Média / Baixa
```
