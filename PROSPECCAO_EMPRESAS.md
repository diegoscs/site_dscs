## 🔎 COMO PESQUISAR NO GOOGLE MAPS

### Passo 1: Categoria
```
Abra Google Maps
Pesquise: "Dentistas São Paulo" (ou outra categoria)
```

### Passo 2: Filtro por Localização
```
Maps mostra uma lista
Clique em cada uma
Anote:
- Nome
- Telefone
- Email (se tiver no perfil)
- Endereço
- Se tem site (verá link azul)
```

### Passo 3: Identificar SEM SITE
```
Sinais de que NÃO TEM SITE:
❌ Só tem WhatsApp na descrição
❌ Nenhum link aparente
❌ Descrição muito curta/vazia
❌ Fotos antigas
❌ Sem avaliações recentes

✅ ESSES SÃO SEUS ALVOS!
```

### Passo 4: Extrair Dados
```
Copie em um lugar:
- Nome: 
- Categoria:
- Telefone:
- Email:
- Endereço:
- Bairro:
- Observação: [ex: Sem site, só WhatsApp]
```
